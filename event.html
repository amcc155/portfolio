I211 Event Management Project

Project Overview

This project is a event management system that allows users to add, edit, and view events thorugh the power of csv files and a database. In this project, I used Python on the backend and Bootstrap on the front-end to create a usable site.

Phase 1

The first phase of the project was strictly powered by csv files. I created a csv file for events, venues, and people. I then used flask to get the data into a list of dictionaries, and used Bootstrap and Jinja to format the data on the front end.

Sample Code

The second part of Phase 1 was adding and editing the events, venues, and people. I created forms in seperate html files and added routes in the app.py so i can retrieve the data. I created a new temporary dictionary to add the user input, then added that dictionary to the total list of dictionaries for each section, and used my created setter function to rewrite the csv file with the new data.
